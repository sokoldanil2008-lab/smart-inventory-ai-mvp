# smart-inventory-ai-mvp
SmartInventory AI
1.Пітч (Pitch)
SmartInventory AI вирішує проблему хаотичного обліку залишків та неочікуваного дефіциту товарів у малого бізнесу. Наша платформа допомагає власникам магазинів та складів автоматизувати рутину за допомогою ШІ, який прогнозує потребу в закупках на основі історії продажів. У результаті користувач отримує оптимізований капітал (гроші не "заморожені" в зайвому товарі) та впевненість, що ходові позиції завжди в наявності.
2. Ядро ідеї (MVP Scenario)
Ключовий сценарій, без якого продукт не має сенсу: автоматичне сповіщення про критичний залишок.
Користувач завантажує список товарів з поточною кількістю.
Система фіксує факт продажу/списання.
ШІ аналізує швидкість збуту і, коли товару залишається на 3 дні роботи, надсилає сповіщення: "Час замовити Товар Х, орієнтовна кількість — 50 шт".
3. Карта стейкхолдерів (Stakeholder Map)
Отримувачі цінності: Власники малого бізнесу та менеджери зі закупівель, які позбуваються страху забути про замовлення.
Оцінювачі результату: Кінцеві клієнти бізнесу (завжди знаходять товар на полиці) та фінансові контролери (бачать зменшення збитків від простою).
Впливові сторони: Розробники (обмежені складністю інтеграції ШІ-моделей) та постачальники даних (API облікових систем), які диктують швидкість оновлення інформації.
4. Мета першої ітерації та ознаки успіху
Мета: Запустити робочий цикл "Облік — Аналіз — Сповіщення" для однієї категорії товарів.
Ознаки успіху:
Користувач витрачає менше ніж 2 хвилини на внесення даних про надходження товару.
Система видає прогноз закупівлі з точністю +/- 10% від реальної потреби протягом тижня тестування.
Головний сценарій (сповіщення про дефіцит) спрацьовує без технічних збоїв 7 днів поспіль.
