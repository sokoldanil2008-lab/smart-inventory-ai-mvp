#  SmartInventory AI: MVP Prototype

Это работающий прототип системы управления складом на базе ИИ.

### Реализованные функции:
1. **AI Vision Simulation:** API принимает "фото" и возвращает список найденных объектов с коэффициентом уверенности.
2. **Predictive Analytics:** Алгоритм рассчитывает дату истощения запасов на основе текущих остатков.
3. **Swagger API Docs:** Автоматическая документация всех эндпоинтов.

### Как запустить:
1. Клонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Запустите сервер: `uvicorn app.main:app --reload`.
4. Перейдите по адресу `http://127.0.0.1:8000/docs` для тестирования функций.
