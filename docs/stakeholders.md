
## Карта стейкхолдеров (Stakeholder Map)

Для успеха проекта мы выделили ключевые группы влияния:

### 1. Внутренние (Internal)
* **Product Owner / Management**: Стратегическое развитие и ROI.
* **AI/ML Team**: Разработка моделей прогнозирования и Computer Vision.
* **DevOps & Infrastructure**: Масштабируемость системы в облаке.
* **Sales & Marketing**: Продвижение продукта на рынке B2B.

### 2. Внешние (External)
* **Клиенты (Retail/Logistics)**: Владельцы складов и торговых сетей.
* **Конечные пользователи**: Кладовщики, операторы, менеджеры по закупкам.
* **Поставщики данных**: Провайдеры ERP-систем (интеграция с SAP, Odoo).
* **Инвесторы**: Финансирование этапов масштабирования.

---

##  Ключевые функции

-   **AI Demand Forecasting**: Предсказание того, какой товар закончится, на основе исторических данных.
-   **Vision Inventory**: Автоматический подсчет товаров на полках по фотографии или видеопотоку.
-   **Auto-Ordering**: Автоматическое создание заявок поставщикам при достижении критического остатка.
-   **Dashboard & Analytics**: Наглядная визуализация эффективности работы склада в реальном времени.

---

##  Технологический стек

-   **Backend**: Python (FastAPI / Django)
-   **AI/ML**: PyTorch, OpenCV, Scikit-learn
-   **Database**: PostgreSQL + Redis (кеширование)
-   **Frontend**: React + Tailwind CSS
-   **Deployment**: Docker, Kubernetes, AWS
